 🇮🇳 Culture Connect India

 About the Project

Culture Connect India is a backend project built to bring together the cultural diversity of India on a single platform.
The idea is simple — help users explore local businesses, events, and products from different regions and connect with culture digitally.

While building this project, the main focus was on writing clean code, structuring the backend properly, and using real-world development practices.



 What this project does

* User authentication (secure login & signup)
* Business listings and management
* Cultural events creation and browsing
* Product and category management
* Cart and order functionality
* Notification system
* Image upload using Cloudinary



Tech Stack

* Node.js
* Express.js
* TypeScript
* MongoDB (Mongoose)
* JWT Authentication
* Cloudinary (for image uploads)
* Nodemailer
